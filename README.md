# Rama Lokesh Reddy Penumallu Portfolio

Responsive personal portfolio website built with React, Vite, TypeScript, Framer Motion, and plain CSS.

## Tech Stack

- React
- Vite
- TypeScript
- Framer Motion
- Lucide React
- CSS

## Live Site

- https://prlr-profile.vercel.app/

## Local Development

```bash
npm install
npm run dev
```

## Project Notes

- Includes a sticky responsive navigation bar.
- Contains Hero, About, Skills, Projects, Contact, and Footer sections.
- Uses scroll-triggered reveals, a parallax hero accent, and reduced-motion support.
- Showcases real projects, competitive programming highlights, and contact links.
